# Customer Segmentation with K-Means

Objective: Segment customers based on purchasing behavior using K-Means.

How to run:
1. Create & activate venv:
   - python -m venv venv
   - venv\Scripts\activate
2. Install packages:
   - pip install -r requirements.txt
3. Start Jupyter:
   - jupyter lab
4. Open `customer_segmentation.ipynb` and run cells.

Data:
- Place your CSV as `data/customers.csv` (optional).
- If no CSV found, notebook will generate a sample dataset.

Deliverables:
- Notebook with clustering results
- Elbow plot
- Cluster scatter plots
- `clustered_customers.csv` (saved)
